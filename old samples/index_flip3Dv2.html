This a simple JS script (based on JQuery) that create a slideshow with all elements in a given div.

The script simply assing the folliwing classes to elements according with a given time interval:

  • slide: to all elements
  • current: to the current element
  • prev: to the previous element (none at the beginning)
  • next: to the next element

It is up to you to define positions, property and animations via css. You can create horizontal and vertical slides, 3D flips, fade and/or zoom effects, etc.
